# Parse Server Express

### Installing 

```bash
$ npm install
```

### Starting the server

```bash
$ npm start
```

### Environment Variables

Environment Variables are stored in the .env file.

```bash
DATABASE_URI=
MONGODB_URI=
CLOUD_CODE_MAIN=
APP_ID=
MASTER_KEY=
SERVER_URL=
PUBLIC_SERVER_URL=
PARSE_MOUNT=
PORT=

MAILGUN_FROM_ADDRESS=
MAILGUN_DOMAIN=
MAILGUN_API_KEY=
```
